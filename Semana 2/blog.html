¿Qué es Emmet?

Emmet es una herramienta que permite escribir código HTML y CSS de manera más rápida mediante abreviaciones que se expanden automáticamente en fragmentos completos de código.

¿Cómo funciona?

Funciona interpretando expresiones cortas con una sintaxis especial, y luego las expande presionando Tab.

div.container>ul>li*3

Resultado:

<div class="container">
  <ul>
    <li></li>
    <li></li>
    <li></li>
  </ul>
</div>

Símbolos principales

Símbolo Significado Ejemplo
> Hijo div>p
+ Hermano h1+p
* Multiplicación li*5
# ID div#main
. Clase div.box

Conclusión

Emmet es una herramienta poderosa que te ahorra tiempo y mejora la productividad al escribir código HTML/CSS. Ideal para desarrolladores frontend.
