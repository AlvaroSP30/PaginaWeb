Introducción a CSS

CSS (Cascading Style Sheets, o Hojas de Estilo en Cascada) es un lenguaje de hoja de estilos que se utiliza para describir la presentación de un documento escrito en un lenguaje de marcado, como HTML. CSS separa el contenido de la presentación visual de una página web, lo que permite a los diseñadores controlar la apariencia de múltiples páginas web desde un solo lugar.

1. Conceptos Básicos de CSS

Los fundamentos de CSS son la base sobre la cual se construye cualquier diseño web.

a. Sintaxis de CSS

Una regla CSS básica consta de un selector y un bloque de declaraciones.

  • Selector: Apunta al elemento HTML que quieres estilizar (ej., p, h1, .clase, #id).
  • Bloque de Declaraciones: Contiene una o más declaraciones, cada una con una propiedad y un valor, separadas por dos puntos (:), y terminando con un punto y coma (;).
/* Selector h1 */
h1 {
    /* Propiedad: valor; */
    color: blue; /* Define el color del texto */
    font-size: 2em; /* Define el tamaño de la fuente */
}

/* Selector de clase */
.mi-clase {
    background-color: lightgray;
    padding: 10px;
    margin: 5px;
}

/* Selector de ID */
#mi-id {
    border: 1px solid black;
}

b. Formas de Incluir CSS en HTML

Hay tres formas principales de integrar CSS en un documento HTML:

  1. En Línea (Inline Styles): Se aplica directamente a un elemento HTML usando el atributo style. 
    <p style="color: red; font-size: 16px;">Este texto es rojo.</p>
  2. Interno (Internal/Embedded Styles): Se define dentro de la etiqueta <style> en la sección <head>. 
    <!DOCTYPE html>
<html>
<head>
    <style>
        body {
            background-color: lightblue;
        }
        h1 {
            color: navy;
        }
    </style>
</head>
<body>...</body>
</html>
  3. Externo (External Stylesheet): El método más recomendado. Los estilos se escriben en un archivo .css separado y se enlazan al HTML usando la etiqueta <link>. 
    <!-- En tu archivo HTML -->
<head>
    <link rel="stylesheet" href="styles.css">
</head>
    /* En tu archivo styles.css */
p {
    font-family: Arial, sans-serif;
}

c. Selectores Comunes

Los selectores son las "llaves" para apuntar a elementos HTML específicos y aplicarles estilos.

  • Selectores de Tipo (Elemento): p, h1, div, a
  • Selectores de Clase: .nombre-clase
  • Selectores de ID: #nombre-id
  • Selectores Universales: *
  • Selectores Descendientes: div p
  • Selectores de Hijo Directo: ul > li
  • Selectores de Hermano Adyacente: h1 + p
  • Selectores de Hermano General: h1 ~ p
  • Selectores de Atributo: a[target="_blank"]
  • Pseudo-clases: :hover, :active, :focus, :nth-child(n), :first-child.
  • Pseudo-elementos: ::before, ::after, ::first-line, ::first-letter.

d. Propiedades Comunes

Algunas de las propiedades más utilizadas en CSS:

  • Texto y Fuentes: color, font-size, font-family, font-weight, text-align, line-height, text-decoration.
  • Modelo de Caja (Box Model): width, height, padding, border, margin, box-sizing: border-box;.
  • Fondos: background-color, background-image, background-repeat, background-position.
  • Bordes: border-width, border-style, border-color, border-radius.
  • Visualización: display (block, inline, inline-block, none, flex, grid), visibility.
  • Posicionamiento: position (static, relative, absolute, fixed, sticky), top, bottom, left, right, z-index.

e. Cascada, Especificidad y Herencia

  • Cascada: Define cómo se aplican múltiples reglas CSS a un mismo elemento.
  • Especificidad: Un algoritmo que determina qué regla CSS prevalece cuando varias la seleccionan. Un selector de ID es más específico que una clase, y una clase más que un elemento.
  • Herencia: Algunas propiedades (ej., `color`, `font-family`) se heredan de los elementos padres a los hijos, mientras que otras (ej., `margin`, `padding`) no.

2. Conceptos Avanzados de CSS

Estos conceptos permiten crear diseños más complejos, responsivos y dinámicos.

a. Flexbox (Módulo Flexible Box Layout)

Un modelo de diseño unidimensional para organizar elementos en una sola fila o columna. Ideal para distribuir espacio y alinear contenido.

.contenedor-flex {
    display: flex;
    justify-content: space-around; /* Distribuye ítems con espacio alrededor */
    align-items: center; /* Centra ítems en el eje transversal */
    height: 200px;
    border: 1px solid #ccc;
}
.item-flex {
    flex: 1; /* Ocupa el espacio disponible equitativamente */
    padding: 10px;
    background-color: lightcoral;
    margin: 5px;
    text-align: center;
}

b. CSS Grid (Módulo Grid Layout)

Un sistema de diseño bidimensional para organizar elementos en filas y columnas simultáneamente. Ideal para diseños de página completos.

.contenedor-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 columnas de igual tamaño */
    gap: 15px; /* Espacio entre celdas */
    height: 300px;
    border: 1px solid #ccc;
}
.item-grid {
    background-color: lightgreen;
    padding: 20px;
    text-align: center;
}
/* Un ítem que abarca dos columnas */
.item-grid:nth-child(1) {
    grid-column: span 2;
}

c. Responsive Design (Diseño Responsivo)

Permite que los sitios web se adapten y se vean bien en una variedad de dispositivos (escritorio, tablets, teléfonos).

  • Media Queries: Reglas CSS que aplican estilos solo si una condición (ej., ancho de pantalla) es verdadera. 
    /* Estilos por defecto para móviles */
body {
    font-size: 16px;
}

/* Media query para pantallas más grandes (ej. tablets y arriba) */
@media (min-width: 768px) {
    body {
        font-size: 18px;
    }
    .contenedor {
        display: flex; /* Cambia a flexbox en pantallas grandes */
    }
}
  • Unidades Relativas: Usar em, rem, vw, vh, % en lugar de px.
  • Imágenes Flexibles: max-width: 100%; height: auto;.

d. Transiciones y Animaciones

  • Transiciones: Permiten cambiar gradualmente el valor de una propiedad CSS. 
    .mi-boton {
    background-color: blue;
    color: white;
    padding: 10px 20px;
    transition: background-color 0.3s ease-in-out, transform 0.3s ease-in-out;
}
.mi-boton:hover {
    background-color: navy;
    transform: scale(1.05); /* Ligeramente más grande al pasar el ratón */
}
  • Animaciones (@keyframes): Crean secuencias de animación más complejas. 
    @keyframes slideIn {
    from {
        transform: translateX(-100%);
        opacity: 0;
    }
    to {
        transform: translateX(0);
        opacity: 1;
    }
}

.elemento-animado {
    animation-name: slideIn;
    animation-duration: 1s;
    animation-fill-mode: forwards; /* Mantiene el estado final */
}

e. Variables CSS (Custom Properties)

Permiten definir valores reutilizables en un solo lugar y usarlos en todo el CSS, mejorando la mantenibilidad.

:root { /* Selector global */
    --color-primario: #3b82f6;
    --espaciado-card: 2rem;
}

.header {
    background-color: var(--color-primario);
}
.blog-card {
    padding: var(--espaciado-card);
    border-color: var(--color-primario);
}

f. Preprocesadores CSS (Sass, Less, Stylus)

Extienden la funcionalidad de CSS añadiendo variables, anidamiento, mixins, etc. El código se "compila" a CSS estándar.

g. Metodologías CSS (BEM, SMACSS, OOCSS)

Conjuntos de reglas para escribir CSS de forma organizada y escalable, como BEM (Block, Element, Modifier).

h. CSS-in-JS

Enfoque moderno donde los estilos se escriben directamente en JavaScript (ej., Styled Components, Emotion), útil en frameworks como React.
